Noter

2020 - p. t.

Description

Noter is a handy notepad for the Android operating system with additional features that have no analogues. Edit text, insert pictures, sort your notes with albums, download webpages, export data to other devices. Combine these functions in order to reach limitless scenarios of usage: a personal diary, own database of chords (if you are a guitarist, for example), a recipe book and much more. With Noter working with notes and webpages will become much easier!

Privacy policy

In a nutshell, Noter does not collect user data and does not contain embedded advertising. If you still have any questions, you can read the privacy policy.

Functions

  • Create, edit and delete notes in just a couple of clicks.
  • Keep notes in special albums, so everything you need will be always at hand.
  • Search for notes you need by name and content.
  • Adjust the order of notes depending on the date of modification, creation or viewing.
  • Download a copy of any webpage for later viewing without an Internet connection.
  • Insert saved webpages and images into notes.
  • Import all data into a file or export it.
